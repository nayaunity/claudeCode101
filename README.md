# Naya Bere | Portfolio

A personal portfolio website showcasing a developer seeking a position at Anthropic.

**Live Site:** https://nayaunity.github.io/claudeCode101/

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Responsive single-page design
- Smooth scroll navigation
- Mobile-friendly layout
- Clean, minimal aesthetic

## Sections

- **Home** — Hero section with tagline
- **Work** — Skills and experience relevant to AI development
- **Why Anthropic** — Alignment with Anthropic's mission
- **About** — Background, technical toolkit, and contact info

## Local Development

Open `index.html` in your browser — no build step required.
